BU-Stats Collector

ร่วมทดสอบการใช้งาน Application โดยมีค่าตอบแทนจำนวน 200 บาท

BU-Stats Collector เป็นเเอพค้นหาไวรัสในเครื่อง โดยแอพจะตราจสอบการใช้งานแอพพิเคชั่นภายในเครื่องว่ามีผลเสี่ยงต่อการติดไวรัสหรือไม่ เพื่อนำข้อมูลมาวิจัยกระบวนการต่อไป เพื่อที่จะพัฒนาแอพเป็น แอพ anti virus ต่อไป

ร่วมทดสอบการใช้งาน Application ตามเงื่อนไขดังนี้

  1. Install Application ลงเครื่องปกติ เฉพาะ Android เท่านั้น

  2. ลงทะเบียนการเข้าร่วมทดสอบแอพ โดยนำ ID ที่แสดงผ่านหน้าจอแอพนำมาลงทะเบียน >>>ลงทะเบียน<<< เปิดรับเพียง 100 เครื่องเท่านั้น
  3. ข้าร่วมกลุ่มทาง Facebook >>>BU-Stats Collector<<<
  4. ทำการใช้งานตามปกติ โดยระบบจะทำการส่งข้อมูลไปยัง Server ทุกครั้งเมื่อแอพทำงานมีขนาดไฟล์เท่ากับ 2048 kb แสดงผ่านหน้าจอแอพ ***ขนาดไฟล์จะขึ้นอยู่กับปริมาณการใช้งานแอพที่มีการเชื่อมต่ออินเตอร์เน็ต เช่น Facebook,Line
  5. การจ่ายเงิน ต้องมีไฟล์ข้อมูลอยู่บน Server อย่างน้อย 7 วัน สามารถตรวจสอบได้ >>>ตรวจสอบไฟล์<<<

ตัวอย่างแอพ

เงื่อนไข การจ่ายเงิน

การจ่ายเงิน มีอยู่ 2 ช่องทาง

  1. เงินสด จุดรับเงิน bts อนุสาวรีย์ , bts ชิดลม , มหาลัยกรุงเพท รังสิต
  2. โอนเข้าบัญชี จะหักค่าธรรมเนียมตามเงื่อนไขของแบงค์ต่างๆ ยกเว้นธนาคารกรุงเทพในกรุงเทพและปริมลฑล
ทางเราจะจ่ายค่าตอบแทนให้ท่านก็ต่อเมื่อ ID เครื่องของท่านมีข้อมูลอยู่บน Server อย่างน้อย 7 วัน สามารถตรวจสอบได้ สามารถตรวจสอบข้อมูลที่ส่งเข้า Server ของตัวเองได้ที่ เมนู ตรวจสอบไฟล์
